BRIEFING:

* PROGRAM THAT FINDS THE MINIMUM DISTANCE BETWEEN 2 POINTS IN A XY PLANE AND HIGHLIGHTS 
  THESE POINTS IN GNUPLOT

BRANCHES:

* THE FOLDER HAS NO BRANCHES.

MAKEFILES:

* EACH BRANCH HAS ITS OWN MAKEFILES THAT:
 -> COMPILES THE CODE
 -> RUNS THE CODE

FILES ENCLOSED:

* "data.temp" HAS THE INPUTS TO THE MAIN PROGRAMS
* "minimum_dis.c" IS THE ACTUAL PROGRAM 
* "Makefile.txt" IS THE MASTER FILE THAT CAN BE USED TO RUN THE CODE
