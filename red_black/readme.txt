BRIEFING:

* PROGRAM THAT IMPLEMENTS RED-BLACK TREES IN C

BRANCHES:

* THE FOLDER HAS NO BRANCHES.

VERIONS:

* "rb'x'" WHERE x=1,2 ARE THE 2 VERSIONS OF THE CODE
* "final.c" IS THE FINAL CODE OF THE TREE

MAKEFILE:

* EACH BRANCH HAS ITS OWN MAKEFILES THAT:
 -> COMPILES THE CODE
 -> RUNS THE CODE


FILES ENCLOSED:

* "rbt.dot" IS THE DOTDUMP FILE THAT IS USED FOR INPUT FOR GRAPHVIZ
* "Makefile.txt" IS THE MAKEFILE THAT AUTOMATES THE EXECUTION OF THE PROGRAMS 
* "rbt.ps" IS THE DOTTY VISUALIZATION OF THE RED-BLACK TREES
